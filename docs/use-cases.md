# MoneyFlow Use Cases

## Authentication

### Register User
A new user creates an account using first name, last name, email and password.

### Login
An existing user signs in using email and password.

---

## Categories

### Create Category
The user creates a category to classify transactions.

### View Categories
The user views all their categories.

### Update Category
The user edits a category.

### Delete Category
The user removes a category.

---

## Transactions

### Create Transaction
The user registers an income or expense.

### View Transactions
The user views their transaction history.

### Update Transaction
The user edits a transaction.

### Delete Transaction
The user removes a transaction.

### Filter Transactions
The user filters transactions by month, category or type.

---

## Savings Goals

### Create Savings Goal
The user creates a financial goal.

### Add Progress to Goal
The user adds money saved toward a goal.

### View Savings Goals
The user views all savings goals.

### Update Savings Goal
The user edits a goal.

### Delete Savings Goal
The user removes a goal.

---

## Monthly Plan

### Create Monthly Plan
The user creates a monthly financial plan.

### Set Category Limits
The user defines spending limits by category.

### View Monthly Plan
The user views the monthly plan and progress.

### Update Monthly Plan
The user updates the plan.

---

## Dashboard

### View Financial Summary
The user views balance, income, expenses, savings progress and expenses by category.

---

## Financial Insights

### View Recommendations
The system shows recommendations based on the user's spending behavior.
