# MoneyFlow

MoneyFlow is a personal finance web application designed to help users track expenses, manage savings goals, and build better financial habits.

## Purpose

The goal of this project is not only to register transactions, but also to help users understand how they spend their money and make better financial decisions.

## Main Features

- User authentication
- Income and expense tracking
- Custom categories
- Monthly dashboard
- Savings goals
- Spending statistics
- Financial habit insights

## Tech Stack

### Backend
- ASP.NET Core
- MySQL
- Entity Framework Core
- JWT Authentication
- Swagger / OpenAPI

### Frontend
- Vue 3
- Pinia
- Vue Router
- PrimeVue
- Chart.js
